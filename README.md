# ai-paper-reader

A Claude skill for reading and understanding AI research papers deeply — not just summarizing them.

## What it does

- Calibrates to your background before diving in
- Breaks down any paper into: Problem → Contributions → Method → Limitations → Implementation
- Decodes math with plain English + concrete examples
- Labels what's actually novel vs. incremental vs. engineering
- Ends with comprehension questions to test your understanding

## Install

1. Download `ai-paper-reader.skill`
2. Go to Claude.ai → Settings → Skills → Upload
3. Done

## Usage

Trigger it by:
- Uploading a paper PDF
- "Explain the DINOv2 paper"
- "Help me read this: [arxiv link]"
- "Break down the Attention Is All You Need paper"
